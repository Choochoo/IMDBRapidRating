    Add IMDb Cookie

    Live IMDb write-back needs your signed-in Cookie request-header value. Rapid Rater saves it locally to `.env.local` as `IMDB_COOKIE`.

    1. Sign into IMDb in Chrome.
    2. Open DevTools, then the Network tab.
    3. Refresh IMDb and click the main `www.imdb.com` request.
    4. Open Headers, find Request Headers, then copy the full value after `Cookie:`.

    Add TMDB API Key

    TMDB provides reliable posters and synopsis text. Rapid Rater saves the key locally to `.env.local` as `TMDB_API_KEY`.

    1. Create or sign into a TMDB account.
    2. Open TMDB API Getting Started.
    3. Use the Credentials section to get an API key or read access token.
    4. Copy the credential value.
    5. Paste it here and save.
