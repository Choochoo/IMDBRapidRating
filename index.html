    Connect IMDb

    IMDb must be connected before rating. Rapid Rater encrypts this connection in your account and uses it only while its controlled background queue submits or reverses your ratings.

    1. Sign in on IMDb in this browser.
    2. On IMDb, open DevTools, select Network, and refresh the page.
    3. Select the main www.imdb.com request and copy the complete Cookie request-header value.

    TMDB settings

    TMDB provides posters, synopsis text, cast, trailers, title details, and country-specific streaming availability. Rapid Rater saves these settings in your account so they work on your other computers.

    1. Create or sign into a TMDB account.
    2. Open TMDB API Getting Started.
    3. Use the Credentials section to get an API key or read access token.
    4. Copy the credential value.
    5. Paste it here, choose the country whose streaming services you use, and save.

    Use the two-letter country code for the services available where you watch, such as US, CA, GB, or AU.

    Add OpenAI API Key

    AI recommendations need an OpenAI API key. Rapid Rater encrypts the key in your account and never returns it to browser JavaScript.

    1. Create or sign into an OpenAI account.
    2. Open the OpenAI API dashboard and create an API key.
    3. Paste the key here and save.
